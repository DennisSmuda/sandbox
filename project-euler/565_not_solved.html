If $p$ (prime number) is not a divisor of $a$, then $\sigma(a \times p^\alpha) = \sigma(a) + \sigma(a) \times p + \sigma(a) \times p^2 + \cdots + \sigma(a) \times p^\alpha$. Think: if $a$ have divisor set of $\{x, y, z\}$, then $a \times p$ should have divisor set of $\{x, y, z, x \times p, y \times p, z \times p\}$, and so on for $a \times p^\alpha$.

Based on this, $$\sigma(p_1^{\alpha_1} p_2^{\alpha_2} \cdots p_n^{\alpha_n}) = \left(\frac{p_1^{\alpha_1 + 1} - 1}{p_1 - 1}\right) \times \left(\frac{p_2^{\alpha_2 + 1} - 1}{p_2 - 1}\right) \times \cdots \times \left(\frac{p_n^{\alpha_n + 1} - 1}{p_n - 1}\right)$$ 2017 is a prime number, so for $\sigma(p_1^{\alpha_1} p_2^{\alpha_2} \cdots p_n^{\alpha_n})$ to be divisible by 2017, at least one of the $\left(\frac{p_i^{\alpha_i + 1} - 1}{p_i - 1}\right)$ terms need to be divisible by 2017.
