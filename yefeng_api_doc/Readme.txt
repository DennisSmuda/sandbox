这是API函数说明文档模板

注意：中文编码为UTF-8，否则会出现乱码！！

在windows 7下用XeLaTex编译通过，推荐安装CTex或者Texlive，版本挑新的。编译遇到问题的欢迎找我来解决。

apidoc.sty文件定义了模板格式，会时常更新，请留意，并请不要随意修改，但是欢迎任何修改意见和建议。

func_table.tex是函数概要表格的代码，一开始可以使用生成工具自动生成大部分代码，省去很多机械性的工作。
func_detail.tex是函数详解的代码，一开始也可以使用生成工具自动生成。
自动生成两个文件后就可以进入手动编辑了，主要是编辑添加各种描述。当然，也可以全手动写，但是会比较费事。
自动生成工具的使用请看工具目录下的Readme文件。
